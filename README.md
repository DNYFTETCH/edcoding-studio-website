# Edcoding Studio Website

This is the official website for **Edcoding Studio**, a coding education platform based in Nigeria.

## Features
- Responsive design for all devices
- Modern UI with clean layout
- Easy navigation
- Optimized for GitHub Pages

## How to Deploy
1. Clone this repository.
2. Customize the content in the HTML files.
3. Replace the placeholder images in the `assets/images` folder with your own.
4. Push the changes to your GitHub repository.
5. Enable GitHub Pages in your repository settings.
